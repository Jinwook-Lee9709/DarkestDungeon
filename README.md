## 게임 설명

- 4명의 영웅들<br>
원작의 성전사, 노상강도, 역병 의사, 성녀를 구현했고 각각 4개의 스킬을 구현했습니다.<br>
- 복도 탐험 <br>
간단한 좌우이동이 가능하며, 문과의 상호작용으로 전투에 돌입 가능합니다.<br>
- 전투<br>
캐릭터 스킬 사용, 캐릭터 위치 바꾸기 등이 가능합니다.<br>
